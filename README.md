# 🌐 Aplicação Front-end (Réplica de Interface)

Um projeto Front-end moderno, interativo e totalmente responsivo, desenvolvido com o ecossistema **React**. Esta aplicação demonstra a implementação de interfaces complexas e o domínio na configuração de ferramentas de estilização utilitária.

### ✨ Destaques da Arquitetura

* **Estilização Ágil com Tailwind CSS:** Uso de classes utilitárias configuradas via `tailwind.config.js` e `postcss.config.js` para um design system consistente e desenvolvimento rápido.
* **Customização de Build (Craco):** Implementação do *Create React App Configuration Override* (`craco.config.js`), permitindo injetar o Tailwind na configuração nativa do Webpack sem a necessidade de executar um `npm run eject`.
* **Segurança e Ambiente:** Configuração de variáveis de ambiente através do arquivo `.env`.
